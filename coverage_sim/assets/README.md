# Ресурсы сцен (USD)

## Назначение

Сюда помещают **эталонные USD-сцены**, собранные в редакторе Isaac Sim (тротуар, кубы, освещение, при необходимости заготовки под пешеходов), чтобы версионировать локацию вместе с кодом.

## Подключение

1. Сохраните файл сцены из Isaac Sim, например `sidewalk_corridor.usd`.
2. Укажите имя в `coverage_sim/configs/world.yaml` в поле `world.scene_name` (путь относительно того, как `IsaacEnvironment` разрешает сцену — при необходимости скопируйте USD в каталог данных Isaac или задайте абсолютный URL в конфиге, если поддерживается вашей сборкой).
3. Выровняйте **`bounds_xy`** с проходимой областью на сцене и обновите извлечение препятствий или YAML со списком дисков/клеток под новую геометрию (`coverage_sim/env/isaac_env.py`).

## Динамика «пешеходы»

Упрощённые агенты (цилиндры/капсулы) и движение по заданным траекториям можно добавлять скриптом на Python; позиции синхронизируются с тем же этапом, что и роботы покрытия. Подробности — в разделе дорожной карты проекта (концепция сцены).
